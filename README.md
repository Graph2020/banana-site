A site about funny bananas, banana's story, etc

# Technologies ⚛️
* React
* TypeScript
* GSAP
* Three.js
* Tailwind
* Vite
* Next.js

# How to run the project 🏃
* Clone Repo
* Install dependencies: npm install
* Run the development server: npm run dev
* Open in your browser: http://localhost:5173
